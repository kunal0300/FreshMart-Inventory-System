# FreshMart Inventory System

A supermarket management web application developed using ASP.NET Core Razor Pages and SQL Server.  
This system helps supermarkets manage inventory, suppliers, stock availability, product sales, and reporting operations through a secure login-based dashboard.

## System Features

- Secure Admin Authentication Login  
- Product Inventory Management  
- Add, Edit, Delete Products  
- Supplier Management System  
- Sales Transaction Recording  
- Product Search by Name, Barcode, Category and Supplier  
- Stock Availability Monitoring  
- Low Stock Alert Detection  
- Dashboard Analytics Overview  
- SQL Server Database Integration  
- Logout and Session Authentication
