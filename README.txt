FreshMartPortal - different supermarket management website

How to run in Visual Studio:
1. Open FreshMartPortal.csproj in Visual Studio.
2. Open Package Manager Console.
3. Run: Add-Migration FirstFreshMartBuild
4. Run: Update-Database
5. Press Ctrl + F5.

This project uses different names, pages, database context, models, layout and CSS from the previous supermarket website.
